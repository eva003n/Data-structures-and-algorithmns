## Brute force Technique
Serves as the base for solving complex problems where u have a buhch of sulutions and u pick one that solves the problem meets requirements

It might not be efficient but it works

### Why brute force
- Simplicity- easier ti implement and understand
- Completeness: It guarantees finding a solution if one exists.
- Baseline It provides a baseline solution against which more complex algorithms can be compared.

### Why no to use brute force
- The problem size is large.
- Real-time or near-real-time solutions are required.
- More efficient algorithms are available (e.g., dynamic programming, greedy algorithms).
